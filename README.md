# LoL Drafter

**LoL Drafter** is a web tool that helps you make better draft decisions in League of Legends.

Select the champions picked by your team and the enemy team — the tool instantly suggests which champions to pick based on counters and synergies, ranked by effectiveness.

## How it works

- Add up to 5 **allied picks** and 5 **enemy picks**
- Get a ranked list of **suggested picks** that counter the enemies and synergize with your allies
- Each suggestion includes the reason why it works, based on champion abilities and matchup dynamics
